Indoor Navigation System

This project involved designing a robust indoor navigation system using MATLAB. By utilizing the Binary Occupancy Map, we were able to create accurate 2D maps that significantly improved the accuracy of location finding by 25%. Additionally, the system reduced the time required to locate specific areas by 30%, enhancing user experience in large indoor environments such as malls, airports, and hospitals.

Predict Covid-19 Using CT Scan Images

This machine learning project, developed in Python, addressed the critical need for rapid Covid-19 diagnosis. We created a predictor capable of analyzing CT scan images to determine whether they belonged to a Covid-19 patient. The model achieved an impressive 80% accuracy rate, demonstrating its potential as a valuable tool in the early detection and management of Covid-19 cases. This project showcased the power of machine learning in enhancing medical diagnostics during a global health crisis.

Forum for People's Health

This platform facilitated important health discussions and enhanced the overall efficiency of healthcare services.
